# 시간 출력 정의

범주: 수익화
작업자: [민운기]마일스

**[수익화 시간 표시에 대한 규칙 정의]**

- 대상: 수익화 콘텐츠(제한 시간이 존재하는 콘텐츠)
- 표기 방법: 기준값에 따른 변형 출력 방식
- 기준: 일, 시간을 기준으로 출력 방식 변경
- 표기는 **2개 항목으**로 출력
    - : 퀵버튼은 2개의 항목으로 출력
    - : 페이지 UI의 경우, 기본은 2개의 항목으로 진행하되 UI 디자인 성격을 고려하여 3개 항목(월/일/시간, 시간/분/초 등) 적용
    - 3개 항목 적용 시에도 일/시간을 기준으로  가장 작은 단위를 포함하여 변동 출력 (예시 - 일/시/분 —> 시/분/초)
- 1개월이 넘어갈 경우 `n개월 n일`로 표시(단, 개월 수가 넘어가는 수익화 요소는 가급적 적용하지 않을 예정)
- 국가별 표기(예: min 표기를 m으로 표기 등)는 기획에서 locale 조정으로 대응
- 본 조건은 수익화 관련 표기에 한하며, 인게임 콘텐츠는 콘텐츠별 별도 표기 방식으로 적용
- 적용 예시
    - 1개월 이상: n개월 n일
    - 1개월 미만 ~ 1일 이상: n일 n시간
    - 1일 미만 ~ 1시간 이상: n시간 n분
    - 1시간 미만 ~ 1분 이상: n분 n초
