# 일반 상점 판매 패키지 오퍼 UI 규격

작업자: [민운기]마일스

[일반 상점 판매 패키지 오퍼 UI 규격 정리]

- 개념 : 일반 상점에서 판매하는 패키지 상품의 오퍼 UI 에 대한 규격 정의
- 구성 :
    - 타이틀 : 패키지 타이틀 출력
    - 기본 설명 : 패키지 상품에 대한 기본 설명 (후킹 문구 적용)
    - 패키지 상품 모델
    - 종료 방식 : 화면 탭 (별도 x 버튼 없음)

1) 전체적인 프레임은 공통 규격 

2) 스트링과 중앙의 패키지 이미지만 변경

** 단 해당 규격은 일반상점에서 판매하는 상품의 오퍼에 한하며, 기타 전용 오퍼 (축하오퍼, 이벤트 오퍼 등)는 각각의 컨셉에 맞춰 다르게 디자인
