**[무료 & 과금 교체 버튼 UI 규칙]**

- 무료 & 과금 버튼 교체
    - 개념 : 디폴트는 무료이나 액션 시 과금 교체로 변경되는 컨텐츠

- 구성 방식
    - 모양은 각 컨텐츠에 맞춰 다르게 적용
    - 색 부분을 통일화 하여 진행

1. 무료 아이콘
- 색 : 녹색
- 표기 : 무료 (FREE) 출력
