# 작업 방법

*시트 이미지 참고

- **tab_category(상품 분류 탭)** :  상품을 어느 UI 탭에 분류할지 지정하는 항목입니다.
    - **1~4: 일반 상점에서 실제 사용되는 각 UI 카테고리 탭**
    - 11~15: 일반 상점에서 실제 사용되는 각 UI 카테고리 탭
- **order(상품 노출 순서)** : UI 카테고리 탭 기준으로 각 상품이 출력되는 순서를 조정하는 항목입니다.
    - 예시 값이 `106`일 경우:
        - 앞 두 자리 `10`: tab_category 번호
        - 마지막 자리 `6`: 해당 탭 내 노출 순서
    - 노출 순서는 오름차순으로 적용되며, 숫자가 낮을수록 더 앞에 배치됩니다.
        - `1`: 가장 첫 번째 노출
        - `2`: 그 다음 순서로 노출

# 결과 예시

- 기존: 무기 업그레이드 팩 203 / 코스튬 파편 상자 205
- **수정**: 무기 업그레이드 팩 **205** / 코스튬 파편 상자 **203**

⇒ 수정 후 GEAR 탭 진입 시 낮은 순서 번호 기준으로 코스튬 파편 상자 → 무기 업그레이드 팩 순으로 출력됩니다.
