# 보스레이드 교환 상점 - S급 아이콘 표시 기준 및 처리

작업자: [정명진] 지누

# 설명

- 보스레이드 상점 > 교환 상점 내 S급 아이콘 표시 여부를 스태틱 데이터로 제어하는 방법을 설명하는 문서입니다.

### 관련 히스토리 참고

- 기존 S급 아이콘은 `StackableItem > exotic` 등급 이상 아이템에 표시됩니다. 단, 영원석(구 초월석)과 등급이 겹쳐 S급 아이콘이 오표시되는 상황이 발생하여 선택적으로 표시할 수 있게 수정하고자 함.

# 작업 시트

- **`devil_hunter_idle/ShopProductData`**

# 작업 방법

*시트 이미지 참고

- S급 아이콘을 표시할 상품의 `ShopProductData.Icon` 칼럼에 "S"를 입력하여 표기를 제어합니다.
- 해당 리소스는 `CacheResourceManager.GetGradeIcon`을 통해 불러옵니다.

# 결과 예시

- S급 아이콘 표시 : S급 무기, 무기 상자 열쇠 등 `ShopProductData.Icon` 칼럼에 "S" 입력된 상품
- S급 아이콘 미표시 : 초월석 등 동일 등급이나 `ShopProductData.Icon` 칼럼에 값이 없는 상품
