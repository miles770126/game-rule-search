## XP Hero 확률 고지 규칙

▶️ 관련일감 : [[뽑기확률] 확률 정보 페이지 표기개선](https://www.notion.so/32593b2d2573809086dff34c6804a3f1?pvs=21) 

- 기본 표기 방식
    - 등급별 확률은 시트데이터 수치를 직접 고지.
        - 천장으로 올라가는 기대값을 고려치 않고 데이터 수치를 직접 고지
    - 모든 확률은 소수점 셋째 자리에서 반올림하여 표기
        - 해당 내용도 `“본 확률은 소수점 셋째 자리에서 반올림하여 표시합니다”`라고 확률정보표에 텍스트로 고지
- 등급내 개별 표기 방식
    - 등급 내 각 무기 개별 확률 미표기
    - 대신 `“ 각 아이템 확률은 동일합니다”` 텍스트 고지
- 천장 표기 방식
    - 천장이 들어간 등급에 `“N회 미획득시 확정”` 텍스트 고지
- 기타 표기 방식
    - 소원기능등의 특수 소환일 경우, 해당 정보를 해당하는 등급에 직접 텍스트 고지

## 확률 표기 관련 법 규칙 및 타 게임 레퍼런스 (넷마블, 호요버스)

**한국 법 기준에서는 “천장을 포함한 확률”을 따로 계산해 공지할 의무는 없고, 기본 확률(개별 시도 확률)을 공개하면 됩니다.**

다만, **천장(pity)이나*보장 시스템이 있는 경우에는 그 구조를 별도로 명확히 설명해야 합니다.**

아래에 **법 기준 + 실제 업계 운영 기준** 을 나눠서 설명합니다.

1. 법 기준 (2024 시행 확률형 아이템 규제)한국은 **「게임산업진흥에 관한 법률」 개정안**으로 2024년부터 확률형 아이템 확률 공개가 **법적 의무**가 되었습니다.

   핵심 요구사항은 다음입니다.

- 각 아이템이 **획득될 확률**
- **확률 변동 여부**
- **확률에 영향을 주는 구조**

즉 **“개별 뽑기에서 발생하는 확률”을 공개하는 것이 기본 원칙**입니다.

2. 천장 시스템이 있는 경우 (법 해석)천장이 있을 때 법적으로 요구되는 것은 **두 가지**입니다.

① 기본 확률 공개예:아이템확률S급 무기1%A급 무기9%기타90%이 **기본 확률 공개는 필수**

② 천장/보장 시스템 구조 설명 예시 :

- **80회 뽑기 내 S급 무기 1개 확정**
- **50회 이후 확률 증가**
- **특정 횟수 이후 선택 보장**

이러한 **확률에 영향을 주는 시스템은 설명해야 함**하지만❌ **천장을 포함한 "평균 획득 확률" 계산 공개 의무는 없음**

3. 실제 업계 운영 방식 (한국 게임들)현재 한국 주요 게임들은 대부분 다음 방식으로 공지합니다.

예시 (가장 일반적인 방식)**[확률]**등급확률S급1%A급9%**[추가 안내]**

- 80회 내 S급 1개 확정
- 50회 이후 확률 증가

즉

- **기본 확률**
- **천장 구조 설명**

**두 개를 분리해서 공개**4. 왜 천장 포함 확률을 공개하지 않는가천장을 포함한 확률은 **고정값이 아니라 유저 행동에 따라 달라집니다.**

예시 :

- 10회만 뽑는 유저
- 80회까지 가는 유저
- 중간에 나오면 리셋

그래서 법에서도 **기본 확률 + 시스템 설명** 방식만 요구합니다.

5. 주의해야 하는 법적 포인트 (중요)천장 시스템에서 **다음은 반드시 공개 대상입니다.**

공개해야 하는 것

- 천장 횟수
- 확률 상승 여부
- 확률 상승 방식
- 픽업 보장 여부

숨기면 **확률 조작으로 처벌 대상**(최대 2년 이하 징역 또는 2천만원 이하 벌금)6. 정리 (실무 기준)

**확률 고지 방식**✔️ 기본 뽑기 확률 공개✔️ 천장/보장 시스템 설명❌ 천장 포함 평균 확률 공개 의무 없음

💡 **라이브옵스/가챠 설계 기준에서 중요한 팁 하나**

   한국 법 기준에서 **가장 문제가 되는 패턴은 이것입니다.**확률은 1%라고 공지

- 실제로는 **천장 전까지 확률 조정**

이 경우 **확률 변동 미공개 → 위법 가능성 높음**

## **** 넷마블 확률 고지 사례**

# 1️⃣ 넷마블 확률 공지 템플릿 구조

## ① 확률형 아이템 기본 설명

먼저 **가챠 상품 설명 + 확률 적용 방식**을 텍스트로 설명합니다.

예시 템플릿

```
■ 확률형 아이템 안내

- 본 상품은 확률형 아이템입니다.
- 구성품은 동일한 확률로 등장하지 않으며 각 아이템별 확률이 다르게 설정되어 있습니다.
- 확률은 소수점 N자리까지 표시됩니다.
- 확률은 게임 업데이트 및 이벤트에 따라 변경될 수 있습니다.
```

# 2️⃣ 확률 테이블 (핵심)

**아이템별 확률을 표 형태로 공개**

예시 템플릿

| 등급 | 아이템 | 확률 |
| --- | --- | --- |
| SSR | 캐릭터 A | 0.5% |
| SSR | 캐릭터 B | 0.5% |
| SR | 캐릭터 C | 5% |
| R | 캐릭터 D | 20% |
| 기타 | 기타 아이템 | 74% |

특징

- **아이템 단위 확률 공개**
- 또는 **등급 + 개별 아이템**

# 3️⃣ 천장 / 보장 시스템 설명

넷마블은 **확률 테이블과 분리해서 설명**합니다.

예시 템플릿

```
■ 보장 시스템 안내

- 동일 상품을 일정 횟수 이용할 경우 특정 아이템이 확정 지급됩니다.
- 100회 이용 시 SSR 캐릭터 1종을 확정 획득할 수 있습니다.
- 확정 획득 시 카운트는 초기화됩니다.
```

중요 특징

- **천장 확률 계산 없음**
- **시스템 설명만 제공**

## **** 호요버스 확률 고지 사례**

# 1️⃣ 호요버스 확률 공지 구조 (기본 구조)

호요버스는 보통 **5단 구조**로 작성됩니다.

1. 기본 확률
2. 등급별 확률
3. 천장 시스템
4. 확률 증가 구조 (soft pity)
5. 픽업 보장 시스템

---

# 2️⃣ 호요버스 확률 공지 템플릿

## ① 기본 확률 안내

```
Base Probability

5★ Item Base Probability: 0.600%
4★ Item Base Probability: 5.100%
3★ Item Base Probability: 94.300%
```

특징

- **등급 기준 확률**
- 개별 아이템 확률은 따로 설명

---

## ② 보장 시스템 (Hard Pity)

```
Guarantee System

If a player does not obtain a 5★ item within 89 attempts,
the 90th attempt is guaranteed to be a 5★ item.
```

특징

- **정확한 횟수**
- **보장 조건**

---

## ③ Soft Pity (확률 증가)

호요버스가 다른 회사와 다른 부분입니다.

```
Starting from the 74th attempt,
the probability of obtaining a 5★ item gradually increases.
```

즉

- 1~73회 : 기본 확률
- 74~89회 : 확률 증가

---

## ④ 픽업 보장 시스템

```
If the first 5★ obtained is not the featured item,
the next 5★ item is guaranteed to be the featured item.
```

이걸 흔히 **50/50 시스템**이라고 부릅니다.

예

- 첫 5성 → 50% 픽업
- 실패 시 → 다음 5성 100%

---

## ⑤ 통합 확률 설명 (Average Probability)

호요버스는 **평균 확률도 공개합니다.**

예

```
Including guarantee mechanics,
the consolidated probability of obtaining a 5★ item is 1.600%.
```

여기서

- 기본 확률 : 0.6%
- 천장 포함 평균 : 1.6%

이 부분이 **한국 게임들과 가장 큰 차이**

---

# 3️⃣ 실제 호요버스 스타일 통합 템플릿

실제 공지는 거의 이렇게 나옵니다.

```
Base Probability

5★ Item: 0.600%
4★ Item: 5.100%
3★ Item: 94.300%

Guarantee System

If a player does not obtain a 5★ item within 89 attempts,
the 90th attempt is guaranteed to be a 5★ item.

Soft Pity

Starting from the 74th attempt,
the probability of obtaining a 5★ item gradually increases.

Featured Item Guarantee

If the first 5★ obtained is not the featured item,
the next 5★ item is guaranteed to be the featured item.

Consolidated Probability

Including guarantee mechanics,
the consolidated probability of obtaining a 5★ item is 1.600%.
```
