# 퀵버튼 시간표시 UI 형식

범주: 수익화
작업자: [민운기]마일스

**[퀵버튼 시간 표시 UI 형식]**

- 퀵버튼
    - 개념 : 오퍼나 컨텐츠를 다시 열기 위하여 메인화면에 출력되는 버튼 (실행 버튼)
    - 대상 : 출력 제한시간이 존재하는 수익화 요소

- 구성 방식
    - 출력 시점 : 오퍼가 출력이 되거나 온보딩이 진행된 시점에 메인화면에 출력
    - 소별 방식 : 제한 시간이 되면 사라짐 (사라지는 별도의 연출은 없으며, 버튼이 사라졌을 때 메인화면의 배치 위치는 배치순번에 따라 다음 차수의 버튼이 해당 위치로 출력된다)
    - 구성 : 아이콘 + 남은 시간
        - 아이콘 하단에 시간 출력 : 시간 출력은 ‘[시간 출력 정의](%EC%8B%9C%EA%B0%84%20%EC%B6%9C%EB%A0%A5%20%EC%A0%95%EC%9D%98%2032093b2d257380ccaabbc091a6f5a39f.md) ‘ 에 따라 정해진 기준에 맞춰 적용
        - 시간표시 앞쪽에 시계 마크 디폴트로 출력
