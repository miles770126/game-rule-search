# 수익화 오퍼 & 상품 설정 규칙

작업자: [민운기]마일스

수익화 오퍼 & 상품 설정 규칙

| 상황 | 세부 규칙 |
| --- | --- |
| ON/OFF 처리 | 모든 오퍼 및 상품의 경우, RemoteConfig를 이용하여 게임 내 ON/OFF 처리가 되어야 함 

조건) 적용 상품 : 상점에서 판매하는 상품, 구매 유도 오퍼, 별도 상품 판매 오퍼

예외사항) Hero Step 과 같은 컨텐츠 내부에 연계된 상품의 경우 컨텐츠 자체의 ON/OFF 로 적용
 |
| 클라이언트 접속 시 오퍼 출력 처리 | 클라이언트 접속 시 출력되는 오퍼의 경우 아래와 같은 조건을 적용

  1. 앱 오픈 시 매일 최초 세션 1회 출력
  2. 상품 오퍼의 경우 해당 상품의 구매 시 출력이 스킵됨 (출력되지 않음)
      ** 단, 여러 구매횟수가 존재하는 상품의 경우 구매제한횟수가 남아 있을 경우 다 소진되기 전까지 위 1번 조건에 맞춰 출력됨 |
