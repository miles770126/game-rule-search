# 상품 보상 골드 (다이나믹 밸런스 수치 적용)

작업자: [민운기]마일스

상품 보상에서 ‘골드의 유저 레벨 기준 효율’ 적용 시 적용 방식

- 개념 : 현재 Static data 로 골드 재화의 유저 레벨 별 1달러 지급양이 정해져 있음. 이를 이용하여 상품 보상을 제공할 때 사용하는 규칙

- Stactic sheet : ShopDynamicReward
- 계산 방법
    - data 상 Amount1 컬럼의 수치가 1달러 기준 골드 재화양
    - 상품의 전체 단가를 기준으로 골드가 차지하는 보상 비율에 맞춰서 기준 골드량을 이용하여 계산
- 적용 방법 : 레벨 별 data 값 x 환산 비율 금액
- 예시
- 굴림 멀티 오퍼

        ** ShopDynamicReward 의 레벨 당 amount1 골드량을 1달러로 산정

— 1단계는 1.4 달러 어치의 골드량을 제공 – “1단계 골드량(반올림 상수값) = ‘레벨에 따른 amount1’ x 1.5

— 2단계는 1.96 달러 어치의 골드량을 제공 – “2단계 골드량(반올림 상수값) = ‘레벨에 따른 amount1’ x 2

— 3단계는 2.5 달러 어치의 골드량을 제공 – “3단계 골드량(반올림 상수값) = ‘레벨에 따른 amount1’ x 2.5

        

       **. 실적용 (환산 비율 금액은 10000분율로 적용)

— 1단계 : dynamic reward x 1500

— 2단계 : dynamic reward x 2000

— 3단계 : dynamic reward x 2500

** 계산 결과의 애매모호한 부분을 최소화 하기 위해 곱해주는 단위를 1.5, 2, 2.5 로 적용

** (예외) ShopDynamicReward 레벨이 현재 200레벨까지 셋팅 되어 있음

→ 셋팅값 마지막 수치로 이후 레벨 계산 : 예를 들어 201 레벨의 경우에도 200레벨에 셋팅된 amount1 값으로 계산
